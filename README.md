# 基于vue2.0的教学实训资源管理系统

> A Vue.js project

## 主要用到的技术有

``` bash
# html+css+js


# vue2.0+element+vue-router+axios+webpack


# 系统利用mock.js模拟数据、数据请求、实现数据增删改查


# 引用echarts进行图表分析

```
